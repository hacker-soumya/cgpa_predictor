# CGPA Predictor
Predict your CGPA before your actual results are out!!!

## Tech Stack used
- **Frontend** : Streamlit.
- **Ml Model** : DecisionTreeRegressor

Used **Pandas** for Data cleaning and preprocessing

The app has been deployed through Streamlit Cloud for now

You can find the app here
[Click Here](https://predcgpa.streamlit.app/)

*Predict your cgpa now!!*

### Future Plans
- Use FASTApi for backend
- Add a database like MySQL to store users records
